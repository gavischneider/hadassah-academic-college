Instructions

The following format is required to use the program: "{command} {url} {arguments}"

Available commands:

  1. t: Check content type
  2. w: Check for words
  3. i: Check for image
  4. l: Check language
  5. q: Quit the program

Additional Commands

In order to extend the program and add more commands to be executed on URLs, do the following:

  1. Create a new Java Class that implements the 'Checker' interface
  2. Create a constructor that takes in all the arguments required to run your command
  3. In the 'CommandManager' class, add a new case for your command in the switch case
  4. In the 'ValidateInput' class, add the letter of your command to the 'commands' array
  5. In the same class, add a new case for your command to the switch case, making sure to add to it all the appropriate argument checks
